Machine Learning Projects

Protein Family Prediction

Implementation of machine learning (KNN, Decision Trees, Bagging, XGBoost) and deep learning (LSTM network) classification models to predict which families different proteins belong, starting from features available in the Protein Data Bank (PDB).

Machine learning models were trained on a dataset including protein biophysical properties, while the deep learning model was trained using protein amino acids sequences.

The LSTM model performed better than the best machine learning classification model (XGBoost), with an increase of F1-score from 0.66 to 0.75 on the test set.

Life Expectancy Prediction

Applied linear regression (Vanilla, Ridge, Lasso) and ensemble methods (Bagging, Random Forest, XGBoost) to predict the life expectancy of a country given its main health indexes. Data cleaning was initially performed to deal with outliers and to apply different strategies for handling missing values.

The best ensemble model (XGBoost) allowed to reduce MSE up to 31% with respect to the best regression model (Ridge).
